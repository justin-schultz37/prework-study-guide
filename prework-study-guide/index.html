Prework Study Guide

HTML

  • The head element contains information about the webpage
  • The body element represents the visible part of the page
  • The title changes the browser tab title
  • The goal of HTML is to create the framework for the webpage

CSS

  • We covered 3 ways to apply CSS, inline, internal, and external
  • Inline Style allows more customization, but can require more work to change
  • Internal can add style to a broader group directly on a webpage
  • External allows you to store style instructions on a different file and apply them by creating a link in the head of an HTML file.

Git

  • Git provides inline commands to interact with GitHub repository
  • Git is important for maintaining version control by creating branches
  • Git also allows creation and navigation of directories using mkdir and cd respectively
  • Git flow is the process of checking out a repository, doing work, and then merging back into the main branch
